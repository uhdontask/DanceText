## 📄 What is DanceText?

DanceText helps you detect AI-generated fake text images. With technology improving, it’s easier to create text images using AI that look real. DanceText scans these images and highlights signs that they might be fake. Use it if you want to verify text images quickly and easily.

## 🖥️ System Requirements

Before you start, make sure your computer meets these requirements:

## 🖼️ Using DanceText to Detect AI-Counterfeited Text Images

1. **Prepare Your Image**  
Save the text image you want to check to your computer.

2. **Start the App**  
Open DanceText from your desktop or Start menu.

3. **Upload the Image**  
Inside the app, click the "Upload" button. Select your image file from your computer.

4. **Scan the Image**  
After uploading, click “Analyze.” DanceText will scan the image and check for patterns typical of AI creation.

5. **View Results**  
The app will show if the text image appears original or AI-counterfeited. It highlights areas that raise suspicion.

6. **Save or Share Results**  
You can save the report as a file or copy the results to share with others.

## 🛠️ Troubleshooting Common Issues

- **Installer Won’t Run**  
Make sure your system meets the requirements and that you have administrator rights. Try downloading again if the file seems corrupted.

- **App Crashes or Freezes**  
Close the app and open it again. If the problem continues, restart your computer. Update your Windows system to the latest version.

- **Image Upload Fails**  
Check that your image file is not too large (preferably under 10 MB) and is in a supported format.

- **Scan Takes Too Long**  
Large images or complex text might take longer. Try resizing the image before scanning.
